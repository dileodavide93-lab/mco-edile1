# MCO EDILE - Ristrutturazioni d'Elite

Landing page per MCO EDILE, azienda specializzata in ristrutturazioni d'interni chiavi in mano.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS (CDN)

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```
   npm install
   ```
2. Run the dev server:
   ```
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000)

## Build

```
npm run build
```
